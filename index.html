HTML

CSS

  • Cascadin Style Sheets (CSS) is used to style a webpage.
  • CSS defines the colors, aesthetics, visual layout, and organization of the webpage.
  • We will use an exteranl CSS style sheet implement "Seperation of Concerns".
  • A comma is used to combine two selectors into a single CSS rule.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
